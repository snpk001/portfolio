sql1 = 'SELECT date_part(\'year\', age(b.term_start, a.first_term)) as period, count (distinct a.id_bioguide) \
FROM \
(\
    SELECT id_bioguide, min(term_start) as first_term \
    FROM \"congress\".legislators_terms \
    GROUP BY 1 \
    ) a \
JOIN \"congress\".legislators_terms b on a.id_bioguide = b.id_bioguide \
GROUP BY 1 order by 1 \
;'

Q3: What percentage of legislators survived in office for two decades or more after their first term started?¶

Query for answer the question:¶
create view congress.survivor as SELECT first_century ,count(distinct id_bioguide) as cohort_size ,count(distinct case when tenure >= 20 then id_bioguide end) as survived_10 FROM ( SELECT id_bioguide ,date_part('century',min(term_start)) as first_century ,min(term_start) as first_term ,max(term_start) as last_term ,date_part('year',age(max(term_start),min(term_start))) as tenure FROM congress.legislators_terms GROUP BY 1 ) a GROUP BY 1</p>
